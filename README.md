<h1 align="center">Library Management App</a>
**Описание**
Приложение для управления библиотекой. Позволяет добавлять, удалять, редактировать и искать книги, а также изменять их статус. Данные о книгах сохраняются в формате JSON для удобного хранения и последующего использования.

**Основной функционал**
**1. Добавление книги**
Позволяет добавить книгу в библиотеку с указанием следующих данных:

<li>Название</li>
<li>Автор</li>
<li>Год издания</li>
При добавлении книги автоматически присваивается уникальный ID. Проверяется корректность года (число и не отрицательное значение).

**2. Удаление книги**
Позволяет удалить книгу из библиотеки по её уникальному ID. При удалении данные обновляются в JSON-файле.

**3. Поиск книги**
Реализован удобный поиск по библиотеке:

<li>По названию</li>
<li>По автору</li>
<li>По году издания</li>
Исключительно регистронезависимый поиск. Найденные книги отображаются в читаемом формате.

**4. Отображение всех книг**
Выводит список всех книг, отсортированных по их ID. Если библиотека пуста, пользователь получает соответствующее уведомление.

**5. Изменение статуса книги**
Позволяет изменить статус книги на один из двух:

<li>"в наличии"</li>
<li>"выдана"</li>
Изменение производится по уникальному ID книги. Пользователь уведомляется об успешности операции.

**6. Сохранение данных**
Данные о книгах автоматически сохраняются в файл library.json после каждого изменения. При запуске приложения данные загружаются из файла, если он существует.

**7. Сортировка книг**
Книги в библиотеке всегда упорядочиваются по ID для удобства отображения и работы с ними.

**8. Меню управления**
Интуитивно понятное меню позволяет выбирать действия, такие как:

<li>Добавление книги</li>
<li>Удаление книги</li>
<li>Поиск книги</li>
<li>Отображение всех книг</li>
<li>Изменение статуса книги</li>
<li>Выход из приложения</li>

**Возможные ошибки и защита от них**
1. Некорректный ввод года:
Программа уведомляет, если год не является числом или меньше 0.

2. Некорректный ID при удалении или изменении статуса:
Пользователь получает уведомление, если книга с указанным ID не найдена.

3. Несуществующий файл JSON:
При отсутствии файла library.json приложение создаёт новый файл автоматически.
